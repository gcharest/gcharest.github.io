Dans le tableau ici-bas, j'ai rassemblé des valeurs de facteur glycémique trouvés à partir de sources que je considère fiables ou pour des recettes que nous faisons à la maison et pour lesquelles j'ai calculées le facteur.

Certaines données trouvées dans le tableau proviennent de sources externes et sont par conséquent sous leur droit d'auteur respectif. Lorsque c'est le cas, les sources seront indiquées dans le tableau.

IMPORTANT: Les facteurs glycémiques calculés pour les recettes ici-bas sont approximatifs et peuvent varier en fonction des quantités utilisées pour les recettes; Ils sont donc offerts sans garanties d'exactitude. Voir la page Calcul du facteur glycémique de recettes pour plus de détails sur la méthode utilisée pour calculer les valeurs des facteurs.
